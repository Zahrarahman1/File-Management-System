# FILE MANAGEMENT SYSTEM

## Description
This project is a File Management System developed using Python and Tkinter GUI. It provides a simple and user-friendly interface to perform basic file operations such as create, read, write, rename, delete, and list files. The system also supports permission control and automatically manages files in a dedicated folder on the Desktop.

## Features
- Create new text files  
- Read file content  
- Write into files (Append / Overwrite modes)  
- Rename files  
- Delete files  
- Display list of all files  
- Permission control (Read / Read-Write)  
- Simple and user-friendly GUI  

## Functional Requirements
The system allows users to:
- Create new files  
- Read existing files  
- Write data into files (append or overwrite)  
- Rename files  
- Delete files  
- View list of all files  
- Enforce read-only and read-write permissions  
- Automatically create and manage storage folder (FMS) on Desktop  

## Technologies Used
- Python  
- Tkinter (GUI)  
- OS Module (File Handling)  
- Datetime Module  
- Pillow (PIL for icons)  
- PyInstaller (for executable file)  

## File Extensions Used
- .py → Source code  
- .txt → User-created files  
- .meta → File permissions storage  
- .png → Icons  

## How to Run

### Method 1 (EXE - Recommended)
- Open `Release` or `dist` folder  
- Double click `main.exe`  
- No Python installation required  

### Method 2 (Source Code)
- Install Python  
- Install required library:
```

pip install pillow

```
- Run:
```

python main.py

```

## Build Information
- Executable created using PyInstaller  
- Supports standalone execution without IDE  
- Runs directly on Windows system  

## Purpose
This project demonstrates file handling, GUI development, and basic operating system concepts in Python.
